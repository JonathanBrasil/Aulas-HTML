Terceiro Pilar do Front-end: JavaScript

Conceitos JS:

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Nessa aula aprendemos sobre o conceito de funções, eventos e DOM: O java script é uma linguagem orientada a funções: Um conjunto de instruções que executa uma tarefa ou calcula um valor. Os eventos são basicamente um conjunto de ações que são realizadas em um determinado elemento da página web, como passar o mouse sobre uma imagem por exemplo. Document Object Model: è utilizado pelo navegador Web para representuar sua página Web.

Referenciamos as variáveis através das palavras reservadas: 'var', 'let', 'const'.
