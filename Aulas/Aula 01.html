Introdução ao HTML 5

Essa página foi criada para fazer as anotações realizada durante as aulas, dessa forma será possível aplicar os conceitos de forma simultânea às anotações

Para realizar um texto e fazer o parágrafo utilizamos a tag ou elemento "p" dentro do sinal de menor e maior

Pricipais tags:

Obs. todas as tags precisam estar dentro do sinal de menor e maior

TAG DESCRIÇÃO
!-- -- Define um comentario
!DOCTYPE Define o tipo de documento
a Define um hiperlink
article Define um artigo
aside Define o conteudo, além do conteúdo da página
audio Define o conteúdo de som incorporado
b Define o texto em negrito
body Define o corpo do documento html
button Define um botão clicável
div Define uma seção do documento
footer Define um rodapé para um documento ou seção
form Define um formulário HTML para entrada do usuário
h1 até h6 Define Cabeçalhos HTML
head Contém metadados
header Define um cabeçalho para um documento ou seção
hr Define mudança de temática no conteúdo
html Define raiz de um documento HTML
iframe Define um quadro embutido
img Define uma imagem
input Define um controle de entrada
label Define um rótulo para um elemento 'input'
ol Define uma lista ordenada
ul Define uma lista não-ordenada
li Define um item da lista
link Define o relacionamento entre um documento e um recurso externo
meta Define metadados sobre um documento html
nav Define links de navegação
option Define uma opção em uma lista suspensa
p Define um parágrafo
script Define um script do lado do cliente
section Define uma seção em um documentoo
select Define uma lista suspensa
small Define um texto maior
span Define uma seção em um documento
style Define informações de estilo para um documento
table Define uma tabela
tbody Agrupa o conteudo do corpo em uma tabela
tfoot Agrupa o conteudo do rodapé em uma tabela
td Define uma célula em uma tabela
th Define uma célula em um cabeçalho de uma tabela
tr Define uma linha em uma tabela
thead Agrupa o conteúdo do cabeçalho de uma tabela
textarea Define um controle de entrada multilinha
time Define um horário específico (ou data e hora)
title Define um título para o documento
video Define um conteúdo de vídeo incorporado
