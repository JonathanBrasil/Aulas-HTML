Nesse parágrafo estamos testando algumas formtações, de cor, ordenação do texto e tamanho. Usando CSS externo na tag P utilizado ID ou Class.

Nesse parágrafo, está recebendo CSS geral de todas as tags P, configuradas com CSS externo na tag Style

Foi criado um style CSS externo, na pasta assets e depois foi carregado na aba Head onde carrega os metas e dessa forma foi possível refaturar a página. Nessa Aula utilizamos CSS in line, CSS interno e CSS externo. Ao utilzar o CSS externo, a organização fica mais fácil, prático e deixa o HTML mais livre, porém também é possível utilizar as outras duas formas.
